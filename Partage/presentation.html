This plugin offers the game mode TaupeGun 1.8, with the possibility of having SuperTaupe, to spice up your games! 
There is the possibility to link the plugin to a database in order to save game statistics.
The texts are currently only available in French.

The plugin was made for the spigot-1.8.8.8-R0.1-SNAPSHOT-latest version.


► GAME MODE

There are several teams of at least 3 players: after 30 minutes of play (configurable), one or more players per team (configurable) are designated as Mole. Their goal is to win with their teammate Mole. For this, they have a kit, a command to communicate with each other and a command that allows them to reveal to all player, in exchange for which they win a golden apple and regenerate 2 hearts.


► DATABASE (OPTIONAL)

Some of the parties data can be saved in a database :

  • Username, last login and time spent in game by each players
  • Name and color of each team
  • Start and duration of the games
  • Roles (mole/supermole), team, number of players kills and information on its status (alive or dead) of each player in each game

To do this, simply go to the config.yml and enter the database connection information. The 4 tables will be automatically created when the plugin is launched for the first time.

bd:
  host-address:
  database-name:
  user:
  password:


► CONFIGURATION IN GAME (/scenarios)

• Configuration (anvil)

  • TIMERS
  • PVP activation time
  • Minute of the moles announcement
  • Minute of the beginning of the reduction
  • MUR
  • Size before reduction
  • Size after reduction
  • Reduction speed
  • SCENARIOS
  • CutClean
  • Blood Diamond
  • Diamond Limit
  • Lava Limiter
  • No Enderpearl Damage
  • No Nether
  • DROPS
  • Apple drop rate
  • Flint drop rate
  • AUTRES
  • Make scenarios invisible
  • Make the coordinates invisible
  • Number of heart regenerated by golden heads (not functional)
  • Allow players the opportunity to choose their team (not functional)
  • TAUPES
  • Activation of supermoles
  • Number of mole per starting team
  • Number of moles per mole team


• Kits (enderchest)
You can create, modify and delete mole kits.

• Equipes (banner)
You can create and delete teams by choosing their color and name. You can also add and remove players from them by clicking on the team banners.
You have the possibility to create a random distribution of players in the teams already created with the paper at the bottom right of the inventory.


► COMMANDS

• Operator commands :

Command Description
/scenarios Allows you to configure the part.
/start To start the game.
/revive To revive a player (before the moles are announced).
/heal Heals all players.
/g To send an alert in the chat room.
/playerkill Removes a player from the game even if he is disconnected.
/players To see the list of players still in play.


• All the players :

Command Description
/rules To see the configuration of the game (in game).


• Moles :

Command Description
/claim Get his mole kit back.
/reveal To reveal himself to all players as a mole.
/t To send a message to his mole team.


• SuperMoles :

Command Description
/superreveal To reveal itself to all players as a supermole.
/supert To send a message to his supermole team.


• Spectators:

Command Description
/taupelist See the list of traitors (moles and supertaupes).



► OTHERS
  • Golden head (craft like the golden apple) regenerates 4 hearts and gives Absorption 1 for 2 minutes
  • Players can be revived before the moles are announced (see commands)
  • The /reveal and /supereveal drop an apple and regenerate 2 hearts
  • Spectators see the messages exchanged between the moles
  • Arrows above the inventory indicate the direction of the teammates - they disappear when the moles are announced
  • Notch apples turn into golden apples when eaten
