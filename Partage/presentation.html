This plugin offers the game mode TaupeGun 1.8, with the possibility of playing the special SuperTaupe (SuperMole) gamemode, to spice up your games! 
There is the possibility to link the plugin to a database in order to save game statistics.
The texts are currently available in French and English. If you want to contribute to the plugin translation, please contact me (my discord is at the end of the page).

The plugin was made for the spigot-1.8.8.8-R0.1-SNAPSHOT-latest version.

 

► CURRENT ASKS

- The plugin was made for the spigot-1.8.8.8-R0.1-SNAPSHOT-latest version.
- Until the 2.3.2 version, the name of the main world must be "world" (server.properties).
- I advise you to pre-generate the chunks before starting the game (with a plugin or in spectator gamemode, browsing the whole map).

 

► GAME MODE

There are several teams of at least 3 players: after 30 minutes of play (configurable), one or more players per team (configurable) are designated as Moles. Their goal is to win with their Mole teammates. In order to do this, they have a kit, a command to communicate with each others and a command that allows them to reveal themselves to all players, in exchange for which they win a golden apple and regenerate 2 hearts.

 

► DATABASE (OPTIONAL)

Some of the rounds data can be saved in a database :

  • Username, last login and time spent in the game by each and every player
  • Name and color of each team
  • Start time and duration of the games
  • Roles (mole/supermole), team, number of players killed and information on its final state (dead or alive) of each player in each game

To do this, edit the config.yml with the database connection sources. The 4 tables will be automatically created when the plugin is launched for the first time.

bd:
  host-address:
  database-name:
  user:
  password:

• Kits (enderchest)
You can create, modify and delete mole kits.

• Equipes (banner)
You can create and delete teams by choosing their color and name. You can also add and remove players by clicking on the team banners.
You have the possibility to create a random distribution of players in the teams already created with the paper at the bottom right of the inventory.

 

► COMMANDS

• Operator commands :

Command Description
/scenarios Allows you to configure the part.
/start To start the game.
/revive To revive a player (before the moles are announced).
/heal Heals all players.
/g To send an alert in the chat room.
/playerkill Removes a player from the game even if he is disconnected.
/players To display the list of players still in play.

 

• All the players :

Command Description
/rules To display the configuration of the game (in game).
/item Gives the lobby item to the player (add in 2.4.1)

 

• Moles :

Command Description
/claim To get the mole kit
/reveal To reveal himself to all players as a mole.
/t To send a message to his mole team.

 

• SuperMoles :

Command Description
/superreveal To reveal itself to all players as a supermole.
/supert To send a message to his supermole team.

 

• Spectators:

Command Description
/taupelist Display the list of traitors (moles and supermoles).

 

► OTHERS
  • Golden head (craft like the golden apple) regenerates 4 hearts and gives Absorption 1 for 2 minutes
  • Players can be revived before the moles are announced (see commands)
  • The /reveal and /supereveal drop an apple and regenerate 2 hearts
  • Spectators see the messages exchanged between the moles
  • Arrows above the inventory indicate the direction of the teammates - they disappear when the moles are announced
  • Notch apples turn into golden apples when eaten
